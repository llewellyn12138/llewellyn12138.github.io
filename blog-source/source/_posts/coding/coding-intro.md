## Python 编程入门：Hello World

### 1. 简介
Python 是一种解释型、面向对象、动态数据类型的高级程序设计语言。

### 2. 环境搭建
访问 [python.org](https://www.python.org/) 下载并安装最新版本。

### 3. Hello World
打开编辑器，输入以下代码：

```python
print("Hello, World!")
```

保存为 `hello.py`，在终端运行：
```bash
python hello.py
```

### 4. 变量与数据类型
```python
x = 5
y = "Hello"
print(x)
print(y)
```
