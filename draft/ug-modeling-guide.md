# UG NX 建模实战指南

本文档旨在提供一份简洁、高密度的 UG NX 建模参考，重点在于操作细节与效率提升。

## 1. 核心操作逻辑

*   **鼠标操作**：
    *   **中键 (MB2)**：旋转视图（按住）、确认/应用（单击）。
    *   **Shift + 中键**：平移视图。
    *   **滚轮**：缩放视图。
    *   **Ctrl + 中键**：缩放视图（比滚轮更平滑）。
    *   **长按右键**：弹出快捷菜单（包含着色模式、适合窗口等）。
*   **选择意图 (Selection Intent)**：
    *   在拉伸、倒角等特征操作时，注意上边框条的“选择范围”。
    *   **单条曲线 (Single Curve)**：仅选择点击的那一条线。
    *   **相连曲线 (Connected Curves)**：自动选择首尾相连的一串线（常用）。
    *   **切线曲线 (Tangent Curves)**：自动选择所有相切连接的线（倒圆角时极其好用）。
    *   **区域边界曲线 (Region Boundary Curves)**：点击闭合区域内部，自动识别边界（类似于填充）。

## 2. 草图 (Sketch) —— 建模之基

**原则**：先画形状，后标尺寸；先几何约束，后尺寸约束。

### 关键步骤详解
1.  **进入草图**：
    *   选择基准平面（XY/YZ/ZX）或实体表面。
    *   *注意*：尽量选择基准平面，避免后续特征删除导致草图丢失附着面（悬空）。
2.  **轮廓绘制**：
    *   使用 `Profile` (Z) 连续绘制直线和圆弧。
    *   *Tip*：绘制直线时按住左键拖动可直接切换为圆弧。
    *   *Tip*：捕捉点（端点、中点、圆心）会自动高亮，但有时会误捕捉，按住 `Alt` 键可临时关闭自动捕捉。
3.  **几何约束 (Geometric Constraints / C)**：
    *   **相切 (Tangent)**：圆弧与直线连接处必须检查，不相切会导致后续倒角失败。
    *   **重合 (Coincident)**：确保封闭截面的点点重合。
    *   **共线 (Collinear)**：让多条线段处于同一直线上。
    *   **对称 (Symmetric)**：利用中心线做对称约束，减少一半标注工作。
4.  **尺寸约束 (Dimensional Constraints / D)**：
    *   双击尺寸修改数值。
    *   *Tip*：遇到过约束（红色报警）时，先删除冲突的几何约束，而非尺寸。
    *   *Tip*：可以使用表达式，例如 `L=100`, `W=L/2`，方便后期参数化修改。

## 3. 基础特征 (Design Features)

### 拉伸 (Extrude / X)
最常用的命令，看似简单但功能深奥。
*   **布尔运算 (Boolean)**：
    *   `None` (无)：创建独立实体（多实体建模基础）。
    *   `Unite` (求和)：与现有实体合并。
    *   `Subtract` (求差)：挖孔、切槽。
    *   `Intersect` (求交)：保留两个实体的重叠部分。
*   **限制 (Limits)**：
    *   `Symmetric Value` (对称值)：从草图平面向两侧等距拉伸（推荐用于轴对称零件）。
    *   `Until Next` (直到下一个)：自动延伸至接触面，无需计算距离。
    *   `Until Selected` (直到被选对象)：延伸到指定的曲面或平面。
*   **偏置 (Offset)**：
    *   可以直接把单线条草图拉伸成薄壁实体（无需画双线）。

### 旋转 (Revolve)
适用于轴对称零件（如轴、盘）。
*   **截面**：必须封闭，且不能穿过中心轴。
*   **轴**：选择草图中的中心线或坐标轴。
*   *Tip*：草图中绘制一半截面即可，利用中心线做对称轴。

### 孔 (Hole)
不要直接画圆拉伸切除，要用孔特征！
*   **类型**：螺纹孔、沉头孔、埋头孔一键生成。
*   **定位**：直接在面上点点，或者使用草图点。
*   **标准**：内置 ISO/ANSI 标准，自动匹配钻头尺寸。

## 4. 细节处理 (Detail Features)

建议在实体基本形状完成后再进行，避免模型过早复杂化，且容易导致后续修改报错。

*   **边倒圆 (Edge Blend)**：
    *   选择顺序：先竖直边，后水平边（通常），或者先大角后小角。
    *   `G1` (相切) vs `G2` (曲率)：外观件（如汽车外壳）建议使用 G2 连续，光影更顺滑；机械件 G1 即可。
    *   *变半径倒圆*：可以在一条边上设置不同的倒圆半径。
*   **倒斜角 (Chamfer)**：
    *   用于轴孔入口导向或去除毛刺。
    *   `Symmetric` (对称) / `Asymmetric` (非对称) / `Offset and Angle` (距离和角度)。
*   **拔模 (Draft)**：
    *   模具/铸造件必备。注意拔模方向与分模面。
    *   *固定面*：通常选择分模面或底面。

## 5. 同步建模 (Synchronous Modeling)

无参建模神器，用于修改没有参数的模型（如导入的 STP/IGES），或者快速修改不想翻历史记录的特征。

*   **移动面 (Move Face)**：直接拖动面调整位置/角度。
*   **替换面 (Replace Face)**：将一个面替换为另一个曲面形状（极其好用，可用于把平顶改成曲顶）。
*   **删除面 (Delete Face)**：快速去除圆角、孔洞或多余特征（修复烂面常用）。
*   **调整圆角大小 (Resize Blend)**：直接修改死实体的圆角半径。

## 6. 装配基础 (Assembly)

*   **添加组件 (Add Component)**：将零件放入装配环境。
*   **装配约束 (Assembly Constraints)**：
    *   `Touch/Align` (接触/对齐)：最常用，面贴面或轴对轴。
    *   `Center` (中心)：两个物体中心对齐（如销钉在孔中居中）。
    *   `Distance` (距离)：设置固定间距。
*   **移动组件 (Move Component)**：
    *   使用“动态手柄”自由拖动组件。
    *   *碰撞检测*：移动时开启碰撞检测，检查干涉。

---
> **总结**：优秀的建模习惯是“草图简单，特征清晰”。尽量避免在草图中绘制过于复杂的线条，多利用特征操作（如阵列、镜像）来生成几何体。
